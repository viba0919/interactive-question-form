## Install
```
$ git clone https://github.com/lukeed/preact-starter vbakula-interactive-question-form
$ cd vbakula-interactive-question-form
$ npm run build
```


## Development

##### Build assets
```
$ gulp
```
Build assets


##### Lint code
```
$ gulp lint
```
Lint javascript and scss code


##### Clean
```
$ npm run clean
```
Remove all downloaded vendors and generated assets
